Fysiske terninger er for bønder og uteliggere. Vi kan lage en med kode i stedet! Her er et eksempel. Se hvordan terningene endrer seg når du laster nettsiden på nytt:


		
		
    
			
  1. 
				
    Lag en funksjon som heter throwDice()
    
				
    function throwDice() {

}
    
			
    2. 
			
			
  2. 
				
    En terning "genererer" random tall. Det samme kan vi gjøre i JavaScript med den innebygde funksjonen Math.random(). Vi kan bruker output()-funksjonen du lagde tidligere for å se resultatet.
    
				
    function throwDice() {
	var random = Math.random()
	output(random)
}
    
			
    
			
    3. 
			
			
  3. 
			
    Hvis du laster siden på nytt vil du se at tallet endrer seg. Math.random() gir oss et tall mellom 0 og 1, men en terning gir tall mellom 1 og 6. Dette kan vi få til ved å multiplisere Math.random() med 6
    
			
    function throwDice() {
	var random = Math.random() * 6
	output(random)
}
    
			
    
			
    4. 
			
			
  4. 
				
    Nå klarer vi å skrive tall fra 0 til 6. Men, disse tallene har jo masse desimaler! Frykt ei, det finnes en innebygget funksjon som runder av tall oppover, til nærmeste heltall. Denne heter Math.ceil() for ceiling ("tak"). Runde av oppover = "tak", get it?
    
				
    function throwDice() {
	var random = Math.random() * 6
	var result = Math.ceil(random)
	
	output(random)
}
    
			
    
			
    5. 
			
			
  5. 
				
    La oss legge til en liten terning-emoji så det ser kulere ut
    
				
    function throwDice() {
	var random = Math.random() * 6
	var result = '🎲 ' + Math.ceil(random)
	
	output(random)
}
    
			
    
			
    6. 
			
			
  6. 
				
    Så spør pappa om dere skal spille Yatzi. MEN YATZI BRUKER JO MANGE TERNINGER, IKKE BARE EN! Da må vi modifisere koden vår litt.
    
				
    I JavaScript finnes noe som heter "løkker", eller "loops" på engelsk. Det finnes flere typer løkker, men den du har lært før bruker en teller i for å gjenta kode i antall ganger.
    
				
    For eksempel En løkke som skriver "Hei" 10 ganger lager man slik:
    
				
    for (var i=0; i<10; i++){
	output("Hei")
}
    
		
			
    Dette kan vi bruke for å kaste i antall terninger. Vi kan også gi funksjonen vår et argument som forteller hvor mange terninger som skal kastes.
    
			
    function throwDice(numberOfDice) {
	for (var i=0; i<numberOfDice; i++){
		var random = Math.random() * 6
		var result = '🎲 ' + Math.ceil(random)
		
		output(result)
	}
}
    
			
    
			
    7. 
			
			
  7. 
				
    Hvis vi vil vise terningene på samme linje kan vi modifisere koden slik
    
				
    function throwDice(numberOfDice) {
	var totalResult = ''
	
	for (var i=0; i<numberOfDice; i++){
		var random = Math.random() * 6
		var result = '🎲 ' + Math.ceil(random)
		
		totalResult += result + '  '
	}
	
	output( totalResult )
}
